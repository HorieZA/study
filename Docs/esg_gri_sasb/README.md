
## ESG(환경·사회·지배구조) 보고서를 작성할 때 전 세계적으로 가장 널리 사용되는 보고 기준 중 두 가지입니다.

## (1) GRI(Global Reporting Initiative)

> 개요

- **목적:** 조직의 **전반적인 지속가능성 영향**(경제, 환경, 사회)을 이해관계자 중심으로 보고하도록 돕는 **전 세계적 기준**
- **적용 대상:** 모든 산업 및 조직 규모 (기업, 공공기관, 비영리단체 등)
- **관점:** 조직이 **사회와 환경에 미치는 영향** 중심 (impact-based approach)

> 주요 특징

| `항목` | `내용` |
| :---: | :---: |
| **기준 체계** | GRI Standards (모듈형 구조) |
| **구성** | - **GRI Universal Standards** (모든 조직 공통)<br>- **GRI Topic Standards** (환경, 사회, 지배구조별 주제 세분화) |
| **중대성(Materiality)** | 조직의 활동이 사회·환경에 **어떤 중대한 영향**을 미치는가에 초점 |
| **보고 목적** | 이해관계자에게 **투명성 확보 및 책임 있는 경영** 증진 |
| **보고 예시** | 온실가스 배출량, 폐기물 관리, 인권, 근로조건, 공급망 관리, 다양성 등 |

> 활용 포인트

- ESG 리포트에서 **정성적·정량적 지속가능성 이슈 전반을 포괄**.
- 글로벌 기업의 **지속가능성 리포트(Sustainability Report)** 표준으로 가장 많이 사용됨.
- 유엔 지속가능발전목표(SDGs)와 연계 가능.
- [GR: 공통 표준 Docs](https://esgdocs.sustainability-report.kr/) 접속

## (2) SASB(Sustainability Accounting Standards Board)

> 개요

- **목적:** 기업이 **투자자에게 재무적으로 중요한 ESG 정보**를 공개하도록 하는 회계기준 중심의 프레임워크.
- **적용 대상:** 산업별 표준 (77개 산업별 기준).
- **관점:** ESG 요인이 **기업 가치와 재무성과에 어떤 영향을 미치는가**에 초점 (investor-focused approach).

> 주요 특징

| `항목` | `내용` |
| :---: | :---: |
| **기준 체계** | SASB Standards (산업별 구분) |
| **중대성(Materiality)** | **재무적 중요성(financial materiality)** 중심 |
| **보고 목적** | 투자자와 자본시장에 **ESG 리스크·기회 관련 정보 제공** |
| **산업 구분** | 예: 은행, 석유·가스, 반도체, 소매업, 통신 등 77개 산업별 기준 |
| **보고 항목 예시** | 에너지 효율성, 데이터 보안, 인권 리스크, 제품 안전, 공급망 관리 등 (산업별로 다름) |

> 활용 포인트

- **기업가치 평가·투자 커뮤니케이션용 ESG 공시**에 적합.
- **기업의 IR 보고서나 연차보고서(Annual Report)** 내 ESG 섹션 작성 시 활용.
- 2021년 이후 **ISSB(국제지속가능성기준위원회)** 가 출범하면서, SASB는 **IFRS S1·S2 기준으로 통합 방향**으로 발전 중.

---

> GRI vs SASB 비교 요약

| 구분 | `GRI` | `SASB` |
| :---: | :---: | :---: |
| **목적** | 조직의 사회적·환경적 영향 공개 | 투자자에게 재무적 ESG 리스크·기회 공개 |
| **관점** | **Impact on society/environment** | **Impact on enterprise value** |
| **적용 대상** | 모든 조직 (범산업적) | 산업별(77개 산업) |
| **중대성 기준** | 사회적·환경적 영향 중심 | 재무적 중요성 중심 |
| **보고 방식** | 정성·정량 지표 혼합 | 정량 지표 중심 (투자자 친화적) |
| **대표 문서** | 지속가능경영보고서 | ESG 투자자 공시 또는 연차보고서 내 ESG 섹션 |

> ESG 보고서 작성 시 활용 전략

| `단계` | `주요 내용` |
| :---: | :---: |
| **① 중대성 평가(Materiality Assessment)** | GRI 프레임워크로 이해관계자 및 사회적 영향 중심으로 평가 |
| **② ESG 이슈 식별** | SASB 기준으로 산업별 재무적 중요 이슈 확인 |
| **③ 데이터 수집 및 검증** | 정량지표(SASB) + 정성정보(GRI) 통합 |
| **④ 보고서 구성** | GRI 구조에 따라 목차·내용 구성, SASB 항목은 부록이나 핵심지표 표로 제공 |
| **⑤ 글로벌 트렌드 반영** | IFRS ISSB(2023~)의 S1, S2 기준과 호환성 검토 (GRI·SASB 통합 방향) |

> 관련 자료 및 참고 사이트

- [**GRI 공식 웹사이트**](https://www.globalreporting.org)
- [**SASB 공식 웹사이트 (Value Reporting Foundation → IFRS로 통합)**](https://www.sasb.org)
- [**ISSB 기준 (IFRS Sustainability Disclosure Standards)**](https://www.ifrs.org/issb/)
- [**IFRS SASB 기준(PDF)**](https://www.ifrs.org/content/dam/ifrs/project/enhancing-the-sasb-standards/ko-sasb-ed-2025-1-proposed-amends.pdf)
- [**2024 LG화학(PDF)**](https://www.lgchem.com/upload/file/sustainability/2024_LGChem_Performance_Data_KOR.pdf)
