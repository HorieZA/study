### 주요 학습 언어 특성
```cmd
1. 프로그래밍 패러다임(programming paradigm) 중 객체지향 프로그래밍 (Class 기반)
2. 객체지향 프로그래밍은 독립된 단위, 즉 "객체"들의 모임을 뜻한다.
3. 객체(Object)란 저장공간에 할당되어 값을 가지거나 식별자에 의해 참조되는 공간 (변수, 자료 구조, 매소드(함수))
```
