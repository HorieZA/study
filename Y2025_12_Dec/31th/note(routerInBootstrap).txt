index.jsx 참조
Bootstrap 적용하기
1. Bootstrap을 설정할 폴더의 터미널에 접속하여 아래 값을 입력
npm install bootstrap bootstrap-icons
2. index.jsx에 아래 import값을 적용
import 'bootstrap/dist/css/bootstrap.min.css'
import 'bootstrap/dist/js/bootstrap.bundle.min.js'

Page2.jsx 참조
1. input은 무조건 "/"로 닫아줘야함
2. readonly는 readOnly로 입력해야함
readOnly="readonly"
3. checked는 아래와 같이 선언이 필요
checked={true} readOnly="readonly"
3-1.다른 checked에는 {false}로 추가해줘야함
