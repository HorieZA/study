React는 module 방식으로 작동되며 기본으로 설정되어있기 때문에 
"type": "module", 설정이 없으며, package.json 파일에 별도로 입력하지 않아도된다.

npx create-next-app@latest
01. NEXT 방식은 내용이 많아서 무거움
02. NEXT 방식 사용 시 자바스크립트를 이용하지만 타입스크립트도 같이 사용하므로 해당 부분도 배워야 함
03. 현시점에서 사용을 꺼려함

npm create vite@latest my-app -- --template react
01. VITE 방식은 가벼움
02. 현시점에서 많이 사용하고 있음

npx create-react-app cra
01. CRA 방식은 v18에서 사용하였으며, v19에서는 지원을 안한다고 하였으나, 사용은 가능
02. 개인적인 생각으로 사용은 가능하나 다음 업데이트가 없다고 보여짐

index.html이 기본적으로 불러와짐

Bootstrap은 React Bootstrap과 기본 Bootstrap이 있으며,
React Bootstrap은 코드 형식으로 이용이 가능함
