팀 GitHub에 접속하여 Popular repositories 클릭 시 해당 Code 페이지로 이동
최초 main만 존재

생성된 프로젝트 연동 방법
Issues 접속 => 우측 Projects에서 연동된 프로젝트 선택 후 Todo 선택
(잠겨있는지 확인 필요)

브런치 연동 방법
Issues 접속 => Development에서 Create a branch 텍스트 클릭 => 레이어팝업에서 이름 입력 및 해당 루트 선택
=> VSCode의 터미널로 이동 => 아래 문구 입력
git fetch origin
git checkout [신규브런치]

VSCode의 터미널에서 해당 프런치 접속 확인 방법
git branch

연동된 GitHub에 React 셋팅하는 방법
GitHub와 최종 연동된 폴더의 터미널 오픈
 => npm create vite@latest [폴더명] 입력
 => react -> javascript -> no -> no(yes로 선택해도 문제 없음)

GitHub 현재 상태 확인 방법
git status
